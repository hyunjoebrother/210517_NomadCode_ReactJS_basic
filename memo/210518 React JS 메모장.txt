210304 Ch1-7 : let, const, var

const : 상수, stable, 변하지 않음, 재선언 금지, 재할당 금지
let : 값 변경 허용, 재선언 금지, 재할당 가능
var : variable 값 변경 허용, 재선언 가능, 재할당 가능

* 변수 첫 사용은 const로 하자
* "..." 는 text 취급, string
* bool -> true/false : text 아님
* float : 소숫점
